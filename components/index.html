Repository system¶

"B2SHARE repository system"

Short- & midterm repository system. In the DDPS this repository system is used for depositing, storing and disseminating datasets.

Archival Bot¶

DANS implementation of the SWORD v2 protocol for automated deposits.

LTP Archive¶

Long-term Preservation Archive A proxy that authenticates clients on behalf of Dataverse, using the basic auth protocol. It is used by dd-sword2 to authenticate its clients by their Dataverse account credentials.
